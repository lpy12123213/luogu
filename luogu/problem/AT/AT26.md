# (iwi)
## 题意翻译
定义对称如下：
1. 空串
2. 一个字母'i'
3. 一个字母'w'
4. 在两个字母('i'或'w')中包含一个对称的字符串，如"iwi","wiw","ii","www"
5. 在括号中包含一个对称的字符串，注意括号有两种形式，1、"()" 2、")("，即"(i)"和")i("均为合法对称字符串

求最少进行几次更改操作，使一个字符串变为对称
## 输入
一个由'i','w','('和')'组成的字符串

## 输出
一个正整数，即最少更改次数

## 样例一
### 输入：
(iwi)
### 输出：
0
### 说明：
改串已为对称，不许更改

## 样例二
### 输入：
ii(((((ww
### 输出：
5
### 说明：
可以更改成ww((i))ww，当然也有其他做法

## 数据范围

输入字符串的长度len&lt;=10
